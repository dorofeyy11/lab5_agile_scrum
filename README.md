# lab5_agile_scrum
В этом репозитории представлена ​​симуляция процесса Scrum с использованием GitHub Issues.
Проект представляет собой простую систему управления задачами, разработанную в соответствии с принципами Agile.
Все требования описываются в виде задач, формирующих бэклог продукта.
Планируется и выполняется один спринт в соответствии с принципами Scrum.
GitHub Issues используются для отслеживания прогресса, приоритетов и завершения задач.
Цель этой работы — понять роли, артефакты и рабочий процесс Scrum на практике.

## Sprint Review

В ходе спринта была реализована базовая функциональность системы.
Были завершены задачи по авторизации пользователей и созданию базового интерфейса.
По итогам спринта получена рабочая основа проекта.
Часть задач, включая документацию и профиль пользователя, не была полностью завершена.

## Sprint Retrospective

Что прошло хорошо:
- Чёткая постановка задач
- Удобное отслеживание прогресса через GitHub Issues
- Понятное планирование спринта

Что пошло не так:
- Некоторые задачи оказались сложнее ожидаемого
- Недостаточно времени на документацию

Что можно улучшить:
- Точнее оценивать сложность задач
- Лучше распределять время внутри спринта
